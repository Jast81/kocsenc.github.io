RaspberryPi


ArchLinuxonian
Since early 2013 I have messed around with Arch and its pitfall wonders. Its simplicity and primal nature made me realize how spoiled many of us are with fancy OS's and gave me an inside look on the core of computing.

Miniature Power

This card size computer is actually what is hosting this website. I've had many little projects which involved a LED display when you got emails to a media hub due to its HDMI capabilities. However, this baby has reached its finesse by being strong enough to host this website using NGiNX. The Pi is running ArchLinux for ARM processors. It proved to be perfectly light weight for a 512MB RAM ARM machine. If you are interested in doing one yourself buy a RaspberryPi and start familiarizing yourself with Linux; try some projects online here and there. The tutorials are really fun but if you have any questions on how I did it there will be a tutorials page soon. For now you can contact me.

Ruby and iOS



Pixel People Simulator in Ruby

This iOS game was created by Chillingo/LambdaMu and really took my attention a couple days after its launch. The game consists of combining people with professions to yield new ones. The combinations were all predetermined and I thought, "Well I can develop a nice program to keep track of what I have unlocked and to see what I can possibly unlock; and help people out." So I did. Using a website and CSV files as a local database I got an awesome prototype in Ruby. If you want to use it or contribute you will need Ruby. My plans for this is to use Tk to make it a large open-source "side app" for people to use while playing Pixel People.
